# Manifesto — LogLine Foundation

Decisão sem prova é opinião. Este manifesto estabelece os princípios de **governança verificável**, a **parceria humano–IA** e o **uso pacífico por padrão**.

- Transparência com prova: ledger público, BLAKE3 + Ed25519.
- Ritos computáveis: proposta → voto → ata → política → diretiva → execução.
- Bilinguismo PT–EN (CPLP como companion language).
